## API Reference 

* Used **Dictionary** to store records
#### Get customers

```http
  GET /customers
```
 

#### Get customer

```http
  GET /customers/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `int` | **Required**. Id of item to fetch |

For **PUT, DELETE** look at in app swagger documentation.

```http
  SWAGGER DOCUMENTATION /swagger/index.html
```
